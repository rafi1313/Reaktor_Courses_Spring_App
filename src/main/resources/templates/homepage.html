Kurs Back–end Developer

Back-end to silnik aplikacji, czyli wszystko to, co działa po stronie serwera, ale bez czego aplikacja nie mogłaby wykonywać określonych operacji. Programiści back-end projektują mechanizmy, na podstawie których funkcjonuje cała aplikacja.

Czego nauczysz się w trakcie kursu?

  • Git: system kontroli wersji
  • MYSQL: projektowanie relacyjnej bazy danych
  • Python: Stworzenie programu w języku Python wykorzystującego bazę danych MYSQL.
  • Java: Zaprojektowanie i stworzenie programu w języku Java implementującego bazę danych MySQL z kontrolą poziomów dostępu w oparciu środowisko graficzne Java FX.
  • Spring: Projekt w oparciu o framework Spring
  • Techniki front-end'owe: HTML, CSS, jQuery, javaScript
